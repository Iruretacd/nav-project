# Proyecto Nav - Android Development

Este repositorio contiene el desarrollo del proyecto **Nav**, una aplicación de Android moderna construida como parte de mi proceso de formación técnica. El enfoque principal es la implementación de interfaces declarativas y arquitecturas escalables.

## 👤 Autor
* **Nombre:** Christian Irureta
* **Rol:** Desarrollador del Proyecto

## 🚀 Tecnologías y Herramientas
El proyecto utiliza las últimas tecnologías del ecosistema Android para garantizar un rendimiento óptimo y un código mantenible:

* **Lenguaje:** [Kotlin](https://kotlinlang.org/) (Versión 2.0.21)
* **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (Arquitectura declarativa)
* **Gestor de Dependencias:** Gradle (Kotlin DSL) con Version Catalogs (`libs.versions.toml`)
* **SDK Target:** Android API 36
* **Compatibilidad:** Java 11

## 🛠️ Configuración Técnica
El proyecto ha sido personalizado y optimizado para resolver conflictos de compatibilidad detectados en fases iniciales, asegurando una compilación estable:

- **Namespace:** `com.christianirureta.nav`
- **Build System:** Gradle 8.13
- **Componentes:** Implementación de Navigation Component para la gestión de flujos entre pantallas.

## 📂 Estructura del Proyecto
* `/app`: Módulo principal de la aplicación.
* `/gradle`: Configuración del sistema de construcción y catálogo de versiones.
* `MainActivity.kt`: Punto de entrada de la aplicación desarrollado íntegramente en Compose.
